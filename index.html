    Hoi, ik ben Joseph Appenteng, een MBO student aan het Media College Amsterdam met een passie voor webdevelopment. Door mijn cursussen en projecten heb ik vooral sterke vaardigheden ontwikkeld in front-end development, terwijl ik ook basiskennis heb van back-end development.     Muse(Leerjaar2)

    In samenwerking met het Amsterdam Museum werkte ik als Software Development student mee aan het project "Nieuw Amsterdam?". Dit project was onderdeel van een tentoonstelling over de gedeelde geschiedenis tussen Amsterdam en New York. Het doel van het project was om één historisch object uit de tentoonstelling tot leven te brengen door middel van een interactieve installatie.

    Project Overzicht

    Dit project is gemaakt door Game Artist (3 studenten), Software Development (5 studenten) en Ruimtelijk Vormgeving (3 studenten). De opdrachtgever van dit project was Amsterdam Museum en de project duurde 9 weken. Mijn rol als software development student was de software voor de installatie ontwikkelen.

    Pokemon(Leerjaar 1)

    Dit project toont mijn vaardigheden in front-end ontwikkeling, API-integratie, en datavisualisatie. De app toont data over mijn favoriet tv-shows, Pokémon en persoonlijke statistieken.
